# Introduction to Supervised Learning Algorithms

## Project Overview

This project was completed as part of my AI & ML Internship (Week 3 - Task 2). The objective of this project is to understand the basics of supervised learning algorithms and their implementation using Python and Scikit-learn.

## Algorithms Implemented

* Linear Regression
* Logistic Regression
* Decision Tree

## Datasets Used

* Diabetes Dataset (for Linear Regression)
* Wine Dataset (for Logistic Regression and Decision Tree)

## Project Files

* `notebooks/supervised_learning_algorithms.ipynb` – Jupyter Notebook containing the implementation of the algorithms.
* `docs/summary_report.pdf` – Summary report of the supervised learning algorithms.
* `data/dataset_source.txt` – Information about the datasets used.
* `requirements.txt` – List of required Python libraries.

## Libraries Used

* pandas
* numpy
* matplotlib
* seaborn
* scikit-learn

## Conclusion

This project provides a basic introduction to supervised learning algorithms by implementing Linear Regression, Logistic Regression, and Decision Tree using Scikit-learn. It helped in understanding the basic workflow of training machine learning models using different datasets.
